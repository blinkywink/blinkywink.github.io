Game Knight is a Rare Outfit in Fortnite that can be purchased in the Item Shop for 1,200 V-Bucks or with Balenciaga Knights for 1,800 V-Bucks. Game Knight was first released in Chapter 2: Season 8 and is part of the Balenciaga Fit Set.

Within LEGO Fortnite Odyssey and other LEGO experiences, Game Knight's appearance is changed to that of a LEGO® minifigure.

Prior to the release of this LEGO® Style in Update v37.10, equipping Game Knight in LEGO Fortnite would instead equip a random default outfit's LEGO® style.
