Event Pass

Overview

The Event Pass (also known as Mini Pass) is a seasonal cosmetic item bundle in Fortnite: Battle Royale, which grants the player rewards in exchange for various event points.

Event Pass items are not exclusive to the event in which they are offered and are likely going to be added to the item shop sometime after the event ends. However, players are unable to access Event Passes from past events.

Currently, the Event Pass consists of two tracks. A paid premium track which costs 1,000 V-Bucks and includes 11 rewards alongside a bonus cosmetic included with the purchase of the track, and a free track, which is accessible to all players.
