Overview

Tilted Towers was a Named Location in Fortnite: Battle Royale, that was added in Chapter 3: Season 1 to the island Artemis. Tilted Towers was a giant city located in the central part of Artemis.

Update v1.6[]

  • Retail Row was added as a part of the first version of the Battle Royale map.

  • Retail Row was sucked into a black hole.

  • Retail Row was added back to Apollo.

Update v27.00[]

  • Eclipsed Estate is no longer accessible due to time travel.

Explanation[]

Infobox[]

  • Name = Name of the location
  • Image = Overview and Top View image
  • Type = Type of the location (Named Location, Landmark, Unnamed Location)
  • Location = Location of location in the map (North of, West of, Southwest of, Southeast of)
  • Island = Athena, Apollo, Artemis, Asteria, Papaya (If present in more than one map insert the order: Athena,Apollo)
  • Landmarks = Landmark within the location
  • Status = Present or Removed
  • Release = Date, Month/DD/YYYY, link to the Update
  • Removed = Date, Month/DD/YYYY, link to the Update
  • Owner = Owner of the location
  • Ower_faction = Faction owner of the location
  • Chests = Number of chests
  • Vending_machines = Number of vending machines
  • Ammo_boxes = Number of ammo boxes
  • Floor_loot = Number of floor loot
  • Produce_boxes= Number of produce boxes
  • Boss = List of bosses of the location
  • Henchmen = List of henchmens of the location
  • Sentry_ais = List of sentries of the location
  • Character = List of characters of the location
  • Wildlife = List of wildlife of the location
  • Vehicles = List of vehicles of the location
  • Previous = Link to previous location
  • Next = Link to next location

Gallery[]

Different tabs:

  • Overview, mainly for images placed in the infobox which include an overview image and a top view image
  • NameOfSubject, name of the photographed subject
  • Promotional Images, for promotional images from Blog or other sources (Instagram, Reddit etc.)
  • Concept Art, for concept art for locations
  • Other, for every other image
