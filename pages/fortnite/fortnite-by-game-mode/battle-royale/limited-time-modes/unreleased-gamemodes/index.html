Unreleased Gamemodes are game modes, Limited Time Modes, or Live Events made by Epic Games that were found in the game files by data-miners. Some modes below could be upcoming, unfinished, or scrapped.

Unfinished Modes

Unfinished Modes[]

"Arnold"[]

Arnold was initally a sandbox mode planned for Chapter 5: Season 1 featuring an open world, explorable lobby, NPCs (Evie, Baba Yaga, "Vendors", etc), Bosses (Huntmaster Saber, The Inkquisitor), wildlife, Dynamic POI[1], cutscenes and crafting.[2] Players would spawn in a Marauder Capsule.

Following suspicions that the gamemode had been scrapped,[3] Arnold seems to have been revamped as an Extraction Shooter.[4]

"Saturn"[]

Saturn was a dungeon crawler mode featuring smaller versions of certain outfits.[5]
