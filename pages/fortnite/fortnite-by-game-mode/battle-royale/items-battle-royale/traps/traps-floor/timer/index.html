Timer is a Device in Fortnite: Creative, added in Update v7.00. It was later removed due to it being outdated.

Function

Function[]

This device displays a countdown timer which can last from 1 to 20 minutes. When the clock reaches 0, a beeping sound will play. By jumping or interacting a vehicle with the button on the device, you can start or pause the timer. You were only able to place it on floors similar to the Launch Pad.

Trap ID

Trap ID[]

The Timer is referred to as Toy Siren in the files.

  • Rare: TID_Floor_Toy_Siren
History

History[]

Update v7.00[]

  • Added Timer device into Fortnite: Creative.

Unkown Update:

  • Removed the Timer device from Creative inventory. Timers that were already placed on islands still function as normal.
