Lootpool Changes (Competitive)

Overview

This is a list of every item not available in competitive playlists in the current season of Fortnite: Battle Royale. Everything not on this list, that is present in the Available Items, is present in Competitive.

Unless otherwise noted, items listed here are also not available in Zero Build competitive.

Last Updated: v24.10 (Chapter 4: Season 2).

Common Reasons for Not Including

Common Reasons for Not Including[]

Here is a list of common reasons for not including items in competitive:

  • Newly added weapons and items that failed evaluation as it can be too powerful for competitive.
  • All Explosive Weapons are too powerful for the current competitive meta, so they are not included.
  • Several mechanics are disabled in competitive to improve server performance. This includes AIs and Fire.
  • Cross-Match features are disabled to maintain balance and not to give players a starting advantage.
