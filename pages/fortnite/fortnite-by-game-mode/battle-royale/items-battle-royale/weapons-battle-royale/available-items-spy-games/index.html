Available Items (Spy Games)

Overview

This article is about the list of available items from Spy Games. For other uses, see Available Items (disambiguation).

Spy Games is a Limited Time Mode featured in Chapter 2: Season 2, and is unique in that it contains many unique weapons and exclusive rarities that only exist within the mode.

Listed below is a comprehensive list of every item available in Spy Games, if the mode is available, with notation if the item is exclusive to the mode.
