Checkpoint Reload

Overview

Checkpoint Reload is a Gameplay Mechanic in Fortnite that was introduced in Chapter 6: Season 4.

Functionality

Functionality[]

Checkpoint Reload allows the player to switch away from a weapon while it is reloading, and equipping it again will put the player back at the latest reload stage, rather than canceling it completely. This only triggers if a weapon's magazine is fully depleted, so it won't activate during partial reloads.

The mechanic originated on the Leadspitter 3000, with the only difference being that it would always activate no matter if the weapon began reloading at an empty magazine or not. It was later adjusted in Chapter 7: Season 1 to its current iteration.
