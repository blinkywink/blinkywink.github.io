Overview

Update v13.30 for Fortnite released on July 21st, 2020.


Battle Royale

Battle Royale[]

Weapons and Items[]

  • Changes
    • Submachine Gun
      • Reduced Epic and Legendary variants' reload time.
    • Flare Gun
      • It is now possible to light campfires with this gun.

Map Changes[]

  • All vehicle props found on the map have been removed.
  • Various Gas Stations have received upgrades to prepare for the Joyride Update.
Save the World

Save the World[]

  • Ventures release on July 27th.
    • An in-depth guide to Ventures can be found here.
  • A new Imposter! Questline is available for the Blasted Badlands Venture Zone.
    • Complete it to earn Dennis as a hero!
