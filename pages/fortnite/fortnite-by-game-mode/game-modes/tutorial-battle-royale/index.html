The Tutorial is a Game Mode in Fortnite: Battle Royale. The Tutorial has its own map and navigates the player through the basics of Battle Royale. It was first released on mobile, followed by the release on all other platforms in Update v22.30. This was reversed in Update v22.40.

This mode is an excellent tool to introduce new players to Fortnite and give a little warm up for people who haven't played the game for a really long time.

Function

Function[]

Tutorial will learn players these mechanics:

  • Looking around
  • Moving
  • Jumping
  • Sliding
  • Mantling
  • Using Weapons
  • Healing
  • Building
  • Searching Chests
