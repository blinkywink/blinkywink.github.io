Overview


The Damage Amplifier Powerup is a Device in Fortnite: Creative, added in Update v11.20.

Function

Function[]

Grants a damage increase to the player who picks it up or triggers it.

Functions
Function Parameters Description
Spawn:void Spawns the powerup into the experience so users can interact with it.
Despawn:void Despawns this powerup from the experience.
SetDuration:void Time:float Updates the Duration for this powerup, clamped to the Min and Max defined in the device. Will not apply to any currently applied effects.
GetDuration:float<transacts> Returns the Duration that this powerup will be active for on any player it is applied to.
GetRemainingTime:float<transacts> Agent:agent If the Agent has the effect applied to them, this will return the remaining time the effect has. Returns -1.0 if the effect has an infinite duration. Returns 0.0 if the Agent does not have the effect applied.
HasEffect:void<transacts><decides> Agent:agent Returns the Agent has the powerup's effect (or another of the same type) applied to them.
IsSpawned:void<transacts><decides> Succeeds if the powerup is currently spawned.
Pickup:void Agent:agent Grants this powerup to Agent.
Pickup:void Grants this powerup without an agent reference. Requires Apply To set to All Players.
SetMagnitude:void Magnitude:float Sets the Magnitude for this powerup, clamped to the Min and Max defined in the device. Will not apply to any currently applied effects. For the Damage Amplifier Powerup, this is the damage multiplier.
GetMagnitude:float<transacts> Returns the current Magnitude for the powerup. For the Damage Amplifier Powerup, this is the damage multiplier.
