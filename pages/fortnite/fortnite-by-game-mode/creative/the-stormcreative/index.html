The Storm is a Gameplay Mechanic in Fortnite: Creative which functions similary to its Battle Royale counterpart but can be customized.

Functionality

Functionality[]

There are two devices that manage The Storm, the Basic Storm Controller and the Advanced Storm Controller. These controllers allow The Storm to be customized. Storm radius and damage can be adjusted with the Storm Controllers. The Advanced Storm Controllers allows multiple storm phases.

End-Game Storm

End-Game Storm[]

There is a separate Storm that used to close in after 4 hours of a Creative match. This Storm dealt 10 damage per second. When players were eliminated by this Storm, they would not respawn, and the game would conclude when only one player remained. As a result of this mechanic, there were no truly endless Creative matches.

This Storm was

    removed

in Update v22.00 and players are instead kicked out of the session once the session timer expires.

Storm -1

Storm -1[]

Even though the End Game storm still doesn't exist, it was kind of reintroduced in v24.01, for Unreal Engine exclusively. The storm never appears in-game, nor in the Unreal Editor. It only appears while creating in the actual client through UE. The game UI reads "STORM -1 OF 0", while in regular creative islands, it is always "STORM 0 OF 3" (unless the storm phases carried over).
