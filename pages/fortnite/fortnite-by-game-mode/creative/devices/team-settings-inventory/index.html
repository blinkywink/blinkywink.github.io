Team Settings & Inventory

Starting Devices, Player, Manger, Settings, UEFN Ready

Memory

20

Overview

The Team Settings & Inventory is Device in Fortnite: Creative, added in Update v7.30 and was converted from a Trap device to a Prop device in Update v23.00.

Function

Function[]

Sets game parameters and starting inventory for one or all teams. Place the device then drop items onto it to register them.

All default values have been Bolded.

Functions
Option Description
End Round When Receiving From Ends the round with the associated team as the winners on receiving a signal from the selected channel. If Team is set to All, the winning team is determined in the usual way.
Events
Option Description
On Team is Out Of Respawns Send Event To Transmits a signal on the selected channel when the team associated with the device out of respawns.
On Enemy Eliminated by Team Member Send Event To Transmits a signal on the selected channel when an enemy has been eliminated by a team member.
On Team Member Eliminated Send Event To Transmits a signal on the selected channel when a team member associated with the device is eliminated.
