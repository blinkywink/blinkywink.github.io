Icon Library

Overview

“
These can be used to show points of interest on the map, lead players to specific areas or objects, or give context to devices like the Capture Area.
”
— V19.10 Blog

The Icon Library is a gallery of Epic-designed icons available for use in certain Devices within Fortnite: Creative. It was first introduced to Creative in Update v19.10 and later became accessible in UEFN with the launch of UEFN on March 22nd 2023.

View the devices in the What links here page.

History

History[]

  • C3S1
  • C3S3

Update v19.10

  • Introduced the Icon Library to Creative.
    • The following Devices support the Icon Library:
      • Accolades
      • Ball Spawner
      • Beacon
      • Capture Area
      • Explosive Device
      • Map Indicator
      • Player Counter

Update v19.20

  • The Objective Device Gallery now supports the Icon Library.

Update v21.30

  • Added 274 new icons to the Icon Library.
    • The Tracker now supports the Icon Library.
