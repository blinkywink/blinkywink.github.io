Overview

Vehicle Spawners allow creators to spawn various vehicles such as Cars, Busses, etc. All vehicles have various customisation options, such as the ability to change their health and prevent certain teams from using them, with Car Spawners being most customisable.

Each Vehicle Spawner will spawn its respective vehicle at the spawn plate's current orientation and location. A preview of how the vehicle will spawn will be displayed while the spawn plate is being placed. If the spawner is Disabled, the Vehicle will spawn after it has been Enabled.

Options[]

Option Value Description
Visible in Game On, Off Determines whether the device is visible during the game. This does affect its collision properties.
Enabled During Phase All, None, Pre-Game Only, Gameplay Only Determines which game phases the device will be enabled in.
Respawn Time Instant, Never, Pick a time Respawns a vehicle that's been destroyed after a selected delay.
Respawn Vehicle when Enabled Yes, No, Only if Needed If this is set to Yes, a vehicle will spawn when the device is enabled. Choosing Only If Needed will not reset an existing vehicle.
Destroy Vehicle when Disabled Yes, No Destroys a spawned vehicle when the spawner is disabled.
Owning Team Any, 1 - 100 Sets the team the device belongs to.
Selected Class None, Any, No Class, Pick a class Determines what class can use this vehicle.

Values for this option are:

  • None: All players, including players with no class assigned, can use the vehicle.
  • Any: Any player with a class assigned can use the vehicle.
  • No Class: Only players with no class assigned can use the vehicle.
  • Pick a class: Pick a class identifier; only players assigned that class can use the vehicle.
Vehicle Health Pick a number, Indestructable Determines how much damage the vehicle can take before it is destroyed.

Receivers[]

Option Value Description
Assigns Driver Add Event Sets the player that instigated the signal as the spawned vehicle's driver
Respawn Vehicle Add Event Spawns a new vehicle after receiving a signal on the selected channel. The existing vehicle will be destroyed before a new vehicle spawns.
Destroy Vehicle Add Event When receiving a signal on the selected channel, the spawned vehicle is destroyed if it exists.
Enable Add Event When a signal is received on the selected channel, the Whiplash spawner is enabled.
Disable Add Event When a signal is received on the selected channel, the Whiplash spawner is disabled.

Transmitters[]

Option Value Description
When Player Enters Vehicle Add Event Transmits a signal on the selected channel when a player enters the spawned vehicle.
When Player Exits Vehicle Add Event Transmits a signal on the selected channel when a player exits the spawned vehicle.
When Vehicle Spawns Add Event Transmits a signal on the selected channel when a vehicle is spawned or respawned.
When Vehicle is Destroyed Add Event Transmits a signal on the selected channel when a vehicle is destroyed.

Car Spawner's have options than other Vehicle Spawner's. You can customize some basic functionality, such as enabling/disabling the Radio, and customizing the Boost capacity of the Vehicle. You can also customize more advanced features such as whether the Vehicle takes damage from Collision, how much damage the Car's explosion does, and how long the Vehicle can stay submerged for before being destroyed.

Chapter 2: Season 6[]

  • Update v16.20: Added the Whiplash, Taxi, Prevalent, Mudflap, and Bear Spawners.
    • All of the previous Vehicle Spawners have been removed, except for the Baller, Driftboard, and X-4 Stormwing, with all except the B.R.U.T.E slated for a future return. This is due to them using an outdated physics engine which caused various issues with the vehicles.
    • All Vehicle Spawners have been converted from a trap to a prop. An option to disable the visibility of the spawner has been added, as well as an option to determine whether the vehicle supports Wraps, if applicable.
  • Update v16.30: Added the Choppa Spawner.
    • Added health options to various Vehicle Spawners.
  • Update v16.40: Added the Boat and Surfboard Spawners.
    • Added the Water Destruction Delay option to the Car Spawners.
  • Update v16.50: Added Explosion and Collision options to the Car Spawners.
