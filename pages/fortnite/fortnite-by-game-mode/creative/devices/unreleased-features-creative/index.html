This Article only contains features which have either been leaked or publicly stated to be coming to Fortnite Creative. For a list of features that are available in other modes, but not Creative, see Missing Features (Creative).

There are many unreleased features in Fortnite Creative. There are unreleased Weapons, unreleased Devices, and even unreleased Islands. There are also partner and collaboration features that are unreleased, such as the Throwing Toy Spawner.

The list below will contain most unreleased features that have been found by data-miners. This list will be updated over time to accurately reflect which features are unreleased or not.
