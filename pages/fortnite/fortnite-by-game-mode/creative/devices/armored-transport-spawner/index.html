All Settings

All Settings[]

Default values are bold. Values that trigger contextual filtering are italic. 
All Options
Option Values Description
Enabled During Phase None, Always, Pre-Game Only, Gameplay Only, Create Only Determines the game phases during which the device will be enabled. Pre-Game includes all phases prior to the start of the game.
Enable Respawn On, Off Determines if the vehicle respawns after being destroyed. If this is set to On, an additional option displays below this one.
Respawn Time Instant, Pick or enter an amount This option only displays if the Enable Respawn option is set to On. Determines the delay after a vehicle is destroyed, before it respawns.
Respawn Vehicle When Enabled No, Only If Needed, Yes If this is set to Yes, a vehicle will spawn when the device is enabled. Only if Needed will not reset an existing vehicle.
Destroy Vehicle When Disabled On, Off Destroys a spawned vehicle when the device is disabled.
Allowed Class No Class, All, Any, Pick or enter a number Determines what class can use this vehicle. Values for this option are:
  • All: All players, including players with no class assigned, can use the vehicle.
  • Any: Any player with a class assigned can use the vehicle.
  • No Class: Only players with no class assigned can use the vehicle.
  • Pick or enter a number: Pick a class identifier; only players assigned that class can use the vehicle.
Visible During Game On, Off Determines whether the vehicle is visible during the game. This does affect its collision properties.
Fuel Consumption On, Off Determines if the spawned vehicle uses fuel. If this is set to On, two additional options are displayed.
Random Starting Fuel On, Off This option only displays if the Fuel Consumption option is set to On. If set to On, the vehicle spawns with a random fuel amount, 25%–80%, and two more options appear. If Off, another option appears.
Starting Fuel 100, Pick a percentage Only displays if Fuel Consumption is On. The percentage of fuel at spawn. 100 means full tank.
Min Random Starting Fuel 0, Pick an amount Only displays if Random Starting Fuel is On. Minimum random fuel percentage the vehicle spawns with.
Max Random Starting Fuel 0, Pick an amount Maximum random fuel percentage the vehicle spawns with.
Fuel Use Multiplier 1.0, Pick or enter a number Only displays if Fuel Consumption is On. Determines how quickly fuel is consumed while driving.
Boost Enabled 'On', Off Determines whether boost is enabled on the vehicle. If set to On, additional options display.
Unlimited Boost On, Off Only displays if Boost Enabled is On. If Off, an additional option displays.
Boost Fuel Use 0.5, Pick or enter a number Displays only if Fuel Consumption and Boost Enabled are On. Controls fuel use while boosting.
Boost Regen Multiplier Default, Pick or enter a number Only displays if Boost Enabled is On. Determines boost meter recharge rate.
Radio Enabled True, False Determines whether the spawned vehicle is able to use the radio.
Spawn with Cow Catcher On, Off Determines whether the spawned vehicle has a cow catcher.
Vehicle Indestructible On, Off Sets whether the spawned vehicle can be damaged or destroyed.
Vehicle Health 800, Pick a number Only displays when Vehicle Indestructible is set to Off. Determines how much damage the vehicle can take before it is destroyed.
Damage Friendly Fire On, Off On means friendly vehicles can damage this device's spawned vehicles on collision.
Damage Other Vehicles On, Off On means this device's spawned vehicles can damage other vehicles on collision.
Allow Damage from Other Vehicles On, Off Determines whether other vehicles cause damage when colliding with vehicles spawned by this device.
Damage Own Vehicle On, Off Determines whether a collision caused by the driver will damage the spawned vehicle.
Max Explosion Delay 1.0, Instant, Pick a delay time Maximum seconds the vehicle has zero health before it explodes.
Lifetime After Explosion 1.0, Instant, Pick an amount of time How long the destroyed vehicle remains after it explodes, before removal.
Explosion Damage to Environment 800, None, Pick an amount of damage Amount of damage dealt to environment objects when the vehicle explodes.
Explosion Damage to Players 800, None, Pick an amount of damage Amount of damage dealt to players when the vehicle explodes.
Explosion Damage to Vehicles 800, None, Pick an amount of damage Amount of damage dealt to other vehicles when the vehicle explodes.
Destroy When Stuck Underwater 'On', Off If set to On, vehicle destroys itself when stuck underwater. Displays Water Destruction Timer. Off disables this.
Water Destruction Timer 5.0 seconds, Pick or enter an amount of time Only displays if Destroy When Stuck Underwater is set to On. Time before the vehicle self-destructs underwater.
Spawn with Vault Yes, No Determines if the vehicle has a vault onboard when it spawns.
Activating Team Any, Pick a team Determines the team this vehicle belongs to.
Allow Driving with Ready Vault Yes, No Determines if players can drive the vehicle while the Vault is ready to be interacted with. Too many such vehicles may cause lag.
Invulnerable While Active Yes, No Determines if the vehicle is invulnerable when the Vault is active.
Spawn With Vault Enabled Yes, No By default, the Vault is not enabled on spawn. Yes sets it active and interactable on spawn.
Loot Pool Reset Small, Medium, Large Determines what size loot pool drops from the Vault when using the default loot pool.
Requires Thermite On, Off Determines if the player needs thermite to begin the vault-opening process.
Weakpoint Passive Damage Per Second Don't Override, 0, Pick or enter an amount Amount of damage the vault's weakpoints take each second. Don't Override starts low and ramps up.
Weakpoint Health 750.0, Pick or enter an amount Damage a weakpoint can take before being destroyed and triggering the next weakpoint.
Number of Weakpoints 5, Pick an amount Number of weakpoints required before vault opens.
Vault Interacting Team Any, Pick a team Determines which team can interact with the device.
Invert Vault Interacting Team Selection On, Off If On, all teams except the one selected in Vault Interacting Team can interact with the device.
Vault Interacting Class No Class, All, Any, Pick a class No Class: only players with no class can interact. All: anyone can. Any: any classed player.
Invert Vault Interacting Class Selection On, Off If set to On, all classes except the one selected in Vault Interacting Class can interact.

Options Description
