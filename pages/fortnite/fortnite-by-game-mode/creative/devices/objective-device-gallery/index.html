Objective Device Gallery

Triggers, Objective

Memory

288

Overview

The Objective Device Gallery is a Gallery of Devices in Fortnite: Creative, added in Update v8.10.

This gallery consists of 10 options to choose from.

Function

Function[]

A selection of destructible items for use as game objectives.

All Settings

All Settings[]

All default values have been Bolded.

Device Settings
Option Value Description
Health 100, 1 - 25000, Invulnerable The amount of hit points this object has.
Owning Team None, Team 1 - Team 100 The Team this device belongs to.
Blast Radius Off, 0.25, 0.5, 1, 2, 3, 5, 10, 20 The radius to destroy all object within, in tiles.
Beacon Off, Arrow, Light Beam, Flare A beacon so players know where this is.
Beacon Color1 Friendly/Enemy, Blue, Red, Orange The color of the beacon
Beacon Scale1 Object Scale, Extra Small, Small, Medium, Large, Extra Large The size of beacon above the objective
Hide Health Bar At 5 Tiles, 1 Tile - 100 Tiles, Never, Always The health bar will show until it is this far away
Show in Objective HUD On, Off
  • On - Show this device in this Objective HUD
  • Off - Don't Show
Objective Identifier None, Choose an Icon Used for distinguishing this device others within the Objective HUD
Custom Objective Text { Custom Text } Custom objective text to show on the HUD, if left as empty it will the Objective Identifier name instead.
Display Damage Numbers On, Off Determines whether damage numbers should be displayed when players deal damage to the device.
Visible During Games On, Off Determines whether the device is visible during the game or not.
Show Destroy Messages On, Off Determines if a message shows in the feed that this device has been destroy.
Critical Notification Threshold None, 5%, 10%, 25% Displays a HUD message warning players of the device's health.
Warning Notification Threshold None, 35%, 50%, 75% Displays a HUD message warning players of the device's health.
Play Audio / VFX Off, On Enables/Disables Audio and VFX.
Score 0 - 1000 Awards this many points when destroyed.
Collision During Games Off, On, Only When Visible Determines whether the device has collision properties during the game.
Collides With Everything, Weapons Only Determines what can collide the device when collision is turned on.
Show Objective Pulse to Instigator Yes, No The Objective Pulse will only appear for the players who activate it. Likewise it will only disappear for the player who activated it.
Show Objective Pulse to Friendly Players Yes, No The Objective Pulse will only appear to Friendly Players indicating the location of the device in relation to the Player.
Show Objective Pulse to Enemy Players Yes, No The Objective Pulse will only appear to Enemy Players indicating the location of the device in relation to the Player.
Device Channel Settings
Option Value Description
Destroy When Receiving From No Channel, 1 - 500 Destroys the objective on receiving a signal from the selected channel.

Turn Off Visibility When Receiving From No Channel, 1 - 500 Makes the device invisible on receiving a signal from the selected channel.
Activate Objective Pulse When Receiving From No Channel, 1 - 500 Enables a pulse at Player location pointing toward this Device.
Deactivate Objective Pulse When Receiving From No Channel, 1 - 500 Disables a pulse at Player location pointing toward this Device.
When Destroyed Transmit On No Channel, 1 - 500 Transmits a signal on the selected channel when the objective is destroyed.
