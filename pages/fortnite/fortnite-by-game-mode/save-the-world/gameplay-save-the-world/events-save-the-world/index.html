Overview

This article is about limited-time Events within Save the World. For other uses of "Events", see Events (disambiguation).

Events are limited-time thematic content packages in Fortnite: Save the World revolving around Event Quests. They may feature new mechanics or introduce new heroes or weapons. Event rewards are earned by completing Event Quests or are purchased from the Event Shop using Gold. There is always one event featured at any given time.

Event Questlines are sets of Quests available during different Venture Seasons. Events and Event Quests are available only for a limited time before being replaced by the next set.
