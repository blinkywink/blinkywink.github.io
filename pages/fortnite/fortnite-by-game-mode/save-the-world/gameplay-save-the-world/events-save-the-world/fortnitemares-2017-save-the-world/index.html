“
It’s the spookiest time of the year, especially in Hexsylvania. A jam packed update with 8 new Heroes, over 20 story quests and new creatures to defend against!
”
— Fortnitemares Announce Trailer

Fortnitemares 2017 was a Limited-Time Event in Fortnite: Save the World, and was the first annual Fortnitemares event. This event featured a new zone, new Halloween-themed heroes, weaponry, and enemy reskins, and a new quest line to complete.
