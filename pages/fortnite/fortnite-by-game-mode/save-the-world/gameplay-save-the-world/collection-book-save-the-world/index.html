Collection Book

Description

Type

Collection System

Mode

Save the World

Introduced

June 25th, 2017

Overview

“
Add items to your Collection Book to level up and earn rewards.
”
— In-Game Description

The Collection Book is a feature in Fortnite: Save the World, that details all Heroes, Defenders, Survivors, and Schematics, except for a select few.

Functionality

Functionality[]

Slotting and Collection Book Levels[]

  • Players are able to slot people and schematics for Collection Book XP.
  • Depending on the rarity of the item, players will gain various amounts of Collection Book XP. When an item is slotted in the book, it will no longer be grayed out. The item can be selected and its details, including perks, stats, and more can be inspected.
  • When an item is slotted, it can still be leveled up and evolved. This provides additional Collection Book XP, though not significantly. It is advisable to wait until the book is nearly filled before maximizing everything.
  • On the top right of the screen, the Page Completion progress is displayed. Certain categories have rewards for Page Completion, as well as for Section Completion. Currently, only the Fortnitemares Costume Party 'Attendees' have this added rewards feature.
  • As players increase their XP, they gain Collection Book Levels. Each level grants small rewards such as Schematics, various types of Account Resources, and Loot Llama Tokens.
  • Players can view upcoming rewards for the next 3 levels, as well as the next major milestone in the book. Some rewards grant a choice between given options, such as one of the four hero classes, two survivor types, etc.

Unslotting[]

  • It is possible to unslot an item from the Collection Book using Legendary Flux.

Researching and Recruiting[]

  • Most Weapon Schematics can be researched using Flux and Weapon Designs, or with Weapon Research Vouchers. Doing so will add a copy of said Weapon Schematic to the Armory, with randomized perks.
  • Most Trap Schematics can be researched using Flux and Trap Designs. Doing so will add a copy of said Trap Schematic to the Armory, with randomized perks.
  • Most Heroes can be recruited using Flux and Training Manuals, or with Hero Recruitment Vouchers. Doing so will add a copy of said Hero to the Armory.
    • This will also unlock any associated Team Perks, if applicable.
  • It is not possible to recruit Defenders and Survivors.
Layout

Layout[]

  • On the left of the Collection Book menu screen is a large list of categories. Upon clicking on one, it opens a dropdown menu with each subcategory within the category. When a subcategory is opened, it displays all of the sections within the subcategory on the right. Each section has specific items of one type.
