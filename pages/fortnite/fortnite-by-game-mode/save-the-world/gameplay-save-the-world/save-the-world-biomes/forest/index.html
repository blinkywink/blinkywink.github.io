Overview


“
A lush green forest.
”
— In-Game Description

The Forest is a biome in Fortnite: Save the World that has large trees, ravines, and caves.

Resources

Resources[]

  • The Forest are recommended for gathering Wood and Stone due to the vast trees and caves.
