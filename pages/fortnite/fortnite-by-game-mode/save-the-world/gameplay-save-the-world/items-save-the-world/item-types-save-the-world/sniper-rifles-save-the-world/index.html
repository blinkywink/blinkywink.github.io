Sniper Rifles (Save the World)

Overview

Sniper Rifles are a subclass of Ranged Weapons in Fortnite: Save the World. The purpose of these weapons is long range combat. Most Sniper Rifles use Heavy Bullets, but also often use Medium Bullets.
