Resources (Save the World)

Overview

“
Mini Llamas, XP Boosts, and other valuable resources.
”
— In-Game Description

Resources are items in Fortnite: Save the World. Each resource can have different functions and are not stored the same. Account Resources are infinite and reserved to everything linked to a player's account and most can be often be found in the Weekly Exchange. Inventory Resources are exclusive items to a player's items and inventory and can be found during missions. Resources most commonly rewarded after a mission is finished.
