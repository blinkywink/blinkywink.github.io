Submachine Guns (Save the World)

Overview

Submachine Guns (or SMGs) are a subclass of Ranged Weapons in Fortnite: Save the World. The purpose of these weapons is to eliminate enemies with a high fire rate within Close Quarters Combat. All SMGs mainly use Light Bullets.
