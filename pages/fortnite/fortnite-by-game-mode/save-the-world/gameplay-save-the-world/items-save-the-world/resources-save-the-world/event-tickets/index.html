Overview


Tickets are an Account Resource and an in-game currency in Fortnite: Save the World.

Functionality

Functionality[]

Tickets can be used to purchase Event Llamas from the "Loot" section of the Llama Shop. Tickets are only available during their respective Venture Seasons. Once a Season ends, any unspent Tickets are automatically spent and converted into associated Event Llamas that can be opened later. As such, players start each Venture Season with 0 Tickets.

Obtained

Obtained[]

Tickets can be obtained by completing Mission Alerts, Event Quests and leveling up in Ventures.
