Hover Turret

Hover Turret

Type

Gadget

Overview

“
Cost: 1 Ammo: Bot Turret. Ammo Regen Time: 150 seconds. Deploys a stationary hover turret that fires 6 rounds per second at enemies within 4 tile range. Turret lasts for 20 seconds
”
— In-Game Description

Hover Turret is a Gadget in Fortnite: Save the World that is unlocked with Commander XP.

Tier 4 Number of charges that can be held for a given ability. Ability Charges +1. 170
