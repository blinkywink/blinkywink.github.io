Overview

“
Sword: Medium. Uses an above average attack speed to deal high sustained damage, with moderate impact. Spectral Slash: Warp to the targeted enemy, dealing heavy damage. May stagger smaller enemies.
”
— In-Game Description

The Spectral Blade is a Legendary Sword in Fortnite: Save the World that can be purchased from the Event Store for 1,680 Gold during the Hexsylvania Venture Season. It is a part of the Fortnitemares Weapons Set.

Functionality

Functionality[]

The Spectral Blade is a strong sword with decent attack speed.
