Unused Items (Save the World)

Overview

This article contains unreleased content that has been scrapped and may not be available in the future.
The content on this page is currently present in the game's files but is not accessible by normal means at this time.

Since the release of Fortnite: Save the World, there have been a number of items that remain unused. These items may have been replaced by another feature, or simply scrapped in development.

Items

Items[]

The following items previously existed in the game, but were never made publicly accessible to players. Developers and data-miners could access these items at one point in the game's lifecycle, but cannot use them in the current version. The items themselves were removed from the files at an unknown date.
