Overview


“
Slice through Husks like a... cold knife… through butter? No that’s not right, but with its fast attack speed combo, snare or stun enemies to leave them frozen solid.
”
— HBSR v27.11

Slicicle is a Legendary Sword in Fortnite: Save the World that is a part of the Holiday Weapons Set. It can be purchased from the Event Store for 1,680 Gold during the Frozen Fjords Venture Season.

Function

Function[]

The Slicicle is a fast-swinging Sword that deals moderate damage.
