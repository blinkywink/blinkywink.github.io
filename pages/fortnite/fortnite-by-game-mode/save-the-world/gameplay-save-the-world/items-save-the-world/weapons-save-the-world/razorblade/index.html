Overview

“
Assault Weapon: Burst. An assault rifle that fires tight bursts that allow for precise targetting in mobile mid-range engagements. Deals good damage, but consumes its magazine very quickly.
”
— In-Game Description

The Razorblade is an Epic and Legendary Assault Weapon in Fortnite: Save the World that is part of the Burst Rifle Schematics. It is the High-End Variant of the Razor.

Function

Function[]

The Razorblade is a hitscan, and medium to long ranged weapon that fires in a 3-round burst with a medium fire rate and damage output. It is an accurate weapon with First Shot Accuracy enabled.
