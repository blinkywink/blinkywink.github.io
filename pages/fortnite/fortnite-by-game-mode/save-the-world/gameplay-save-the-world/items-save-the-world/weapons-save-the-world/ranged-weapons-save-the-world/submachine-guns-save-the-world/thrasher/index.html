Overview

“
SMG. Fires at a very high rate, dealing great damage while consuming a lot of ammo. Good for run-and-gun combat, and accurate enough to fight at mid range when fired in short bursts.
”
— In-Game Description

The Thrasher is a Rare - Legendary Submachine Gun in Fortnite: Save the World that uses Light Bullets and is a part of the Core Game Weapons Set.

Function

Function[]

The Thrasher is a hitscan, automatic, and short to medium ranged weapon with a fast fire rate and deals a moderate amount of damage. It is an accurate weapon with First Shot Accuracy enabled.
