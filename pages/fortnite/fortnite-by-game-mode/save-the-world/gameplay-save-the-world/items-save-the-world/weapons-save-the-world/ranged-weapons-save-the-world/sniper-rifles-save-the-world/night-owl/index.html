Overview

“
Sniper Rifle: Bow - Black Metal. Fires an arrow that drops over time dealing moderate damage and heavy impact. Damage, range, and velocity increase with charge time.
”
— In-Game Description

The Night Owl is a Legendary Sniper Rifle in Fortnite: Save the World that uses Heavy Bullets and is a part of the Black Metal Weapons Set. It can be obtained from Fortnitemares Llamas for 500 Candy.

Function

Function[]

Like all other bows, the Night Owl can be charged to increase its damage (to a max of 1.75x), range and bullet velocity.

Spread Weapon Has 0 Spread
