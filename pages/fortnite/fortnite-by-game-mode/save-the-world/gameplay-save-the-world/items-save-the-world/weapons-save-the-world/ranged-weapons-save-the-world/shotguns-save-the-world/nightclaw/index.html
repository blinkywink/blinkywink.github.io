Trivia

Trivia[]

  • The Nightclaw is modeled after the American Mossberg Maverick 88. Fitted with a folding stock.
    • This is further reinforced by the weapon displaying the text "BOSSMERG 12" under the ejection port. Parodying the real world firearms manufacturers.
    • The Nightclaw, Panther and Longarm Enforcer all share the same base model. With various extra changes to set them apart.
  • The Rare variant of the Nightclaw used to be called Precision Panther.
  • The Epic variant of the Nightclaw used to be called Clipper.
