Overview

“
Assault Weapon: Precision Burst. A burst-fire tactical assault rifle featuring a scope. Good damage and impact at long range, but consumes a lot of ammunition.
”
— In-Game Description

The Deathstalker is an Epic and Legendary Assault Rifle in Fortnite: Save the World. It uses Medium Bullets.

The Deathstalker has a Burst Firing Mode, meaning it shoots a round of 2 bullets every time the trigger is pulled.
