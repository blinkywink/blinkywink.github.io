Overview

“
Shotgun: Double-Barreled. Deals heavy damage and knockback to multiple enemies in a very wide cone. Has extremely limited range, tiny ammo capacity, and a long reload time.
”
— In-Game Description

The Double-punch is a Common - Rare Shotgun in Fortnite: Save the World that is part of the Core Game Weapons Set. The High-End Variant of this weapon is the Thunderbolt.

Stats[]
