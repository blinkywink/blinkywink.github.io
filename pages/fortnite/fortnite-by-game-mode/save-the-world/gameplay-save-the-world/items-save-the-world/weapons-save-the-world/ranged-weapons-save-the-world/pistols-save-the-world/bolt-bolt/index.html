Overview

“
Pistol: Bolt Semi-Automatic. Rapidly fires high-powered bolts that can pierce through multiple enemies.
”
— In-Game Description

The Bolt Bolt is an Epic and Legendary Pistol in Fortnite: Save the World that uses Medium Bullets, and is part of the Core Game Weapons Set.

The Bolt Bolt's low-end variant is the Junkyard Pal.

Functionality

Functionality[]

The Bolt Bolt is a semi-automatic weapon with a moderate fire rate. Its bullets can pierce multiple enemies at once.
