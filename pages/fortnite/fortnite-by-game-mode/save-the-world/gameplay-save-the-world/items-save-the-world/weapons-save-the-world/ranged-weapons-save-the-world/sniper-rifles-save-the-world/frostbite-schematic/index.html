Overview


“
Sniper Rifle: Semi-Automatic. A mid to long range rifle that deals good damage and can be fired quickly. Scoped for additional down sights accuracy.
”
— In-Game Description

The Frostbite is Legendary Sniper Rifle in Fortnite: Save the World that is a part of the Holiday Weapons Set.

Functionality

Functionality[]

The Frostbite is a Semi-Auto Sniper with a magazine size of 10. It fires moderately fast, and deals moderate damage.
