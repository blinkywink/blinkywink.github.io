Overview

“
Sniper Rifle: Bow - Medieval. Fires an arrow that drops over time. Damage, range and velocity increase with charge time.
”
— In-Game Description

The Knightfire is a Legendary Sniper Rifle in Fortnite: Save the World that uses Heavy Bullets and is a part of the Medieval Weapons Set. It can appear in the Weekly Store and be purchased for 1,680 Gold.

The Knightfire's Scavenger Counterpart is the Scrap Shot, which functions identically but has a lower crafting cost and durability.

Function

Function[]

Like all other bows, the Knightfire can be charged to increase its damage (to a max of 1.75x), range and bullet velocity.
