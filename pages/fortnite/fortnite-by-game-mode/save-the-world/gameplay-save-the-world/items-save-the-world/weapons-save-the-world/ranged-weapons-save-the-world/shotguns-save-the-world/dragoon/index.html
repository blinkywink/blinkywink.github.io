Overview


“
Shotgun: Flintlock. Deals heavy damage and knockback to multiple enemies in a very wide cone. Has extremely limited range, tiny ammo capacity, high recoil, and a long reload time.
”
— In-Game Description

The Dragoon is an Epic to Legendary Shotgun in Fortnite: Save the World that is a part of the Pirate Weapons Set.
It is available in the Pirate Llamas during the Scurvy Shoals Venture Season.

Functionality

Functionality[]

The Dragoon is an over-under shotgun that fires two shells at once. It has heavy recoil and extremely short range but deals moderate damage.
