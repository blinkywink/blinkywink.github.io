Overview

This article is about the Save the World weapon. For the Outfit of the same name, see Terminator (Outfit).

“
Assault Weapon: Light Machine Gun. A heavy weapon with high damage, an extreme rate of fire, and a huge magazine size. Very inaccurate unless aiming down sights from a fixed position, and firing in short bursts.
”
— In-Game Description

The Terminator is a Rare - Legendary Assault Weapon in Fortnite: Save the World, that is part of the LMG Rifle Schematics.

Function

Function[]

The Terminator is a hitscan, automatic, and medium to long ranged weapon with a very fast fire rate and deals a high amount of damage. Its accuracy however gets worse when fired for long periods of time with First Shot Accuracy enabled.
