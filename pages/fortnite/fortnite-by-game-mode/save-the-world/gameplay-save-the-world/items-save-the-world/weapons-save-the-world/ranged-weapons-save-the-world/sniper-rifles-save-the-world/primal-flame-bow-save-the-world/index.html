Overview

This article is about the Save the World Schematic. For the Battle Royale Weapon, see Primal Flame Bow.

“
Fast-firing bow with a flaming arrow tip that ignites enemies on hit.
”
— In-Game Description

The Primal Flame Bow is a Legendary Sniper Rifle in Fortnite: Save the World that uses Heavy Bullets and is a part of the Primal Weapons Set.

It can be unlocked by completing the Cracking Crystals Quest in the Hunt the Titan Questline, available year-round.

Function

Function[]

Like all other bows, the Primal Flame Bow can be charged to increase its damage (to a max of 1.75x), range and bullet velocity (allowing it to shoot through multiple enemies). The Primal Flame Bow deals fire affliction damage by default.
