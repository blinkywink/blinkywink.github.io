Overview

“
Explosive Weapon: Cannon - Flintlock: Fires cannon balls that damage and knock back groups of enemies. The cannon balls can bounce before exploding, damaging nearby enemies with each bounce.
”
— In-Game Description

The Bowler is a Legendary Launcher in Fortnite: Save the World that uses Rockets and is a part of the Pirate Weapons Set.
It is available in the Event Shop during the Scurvy Shoals Venture Season.

Function

Function[]

The Bowler is a projectile-based explosive weapon that launches cannonballs that bounce around.

Spread 2.5
