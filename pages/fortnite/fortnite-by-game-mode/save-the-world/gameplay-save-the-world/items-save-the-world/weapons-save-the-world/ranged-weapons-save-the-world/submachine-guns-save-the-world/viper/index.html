Overview


“
SMG. Sprays bullets at an extreme rate, dealing punishing damage. Quick to reload and good for running combat, but ammo hungry and ineffective beyond close range.
”
— In-Game Description

The Viper is a Common - Legendary Ranged Weapon in Fortnite: Save the World that uses Light Bullets.

The Viper is a hitscan weapon. It does not have First Shot Accuracy enabled, and will always fire a random spread of bullets, which increases with continuous fire.
