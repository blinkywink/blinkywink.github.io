Overview


“
Shotgun: Retro Sci-Fi. Mid-ranged energy shotgun that shoots large rings of plasma. Shots pierce enemies.
”
— In-Game Description

The Pulsar 9000 is a Legendary Shotgun in Save the World that is a part of the Retro Sci-Fi Weapons. It was first introduced in Update v9.41.

Functionality

Functionality[]

The Pulsar 9000 has a Semi-Auto Firing Mode, meaning the trigger must be pushed down each time to shoot a round of bullets. The Pulsar 9000 has a unique feature where firing the weapon will shoot large plasma rings that deal AoE Damage.
