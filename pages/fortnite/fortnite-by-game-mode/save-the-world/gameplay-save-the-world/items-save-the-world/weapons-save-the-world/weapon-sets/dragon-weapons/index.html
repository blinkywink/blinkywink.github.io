Overview


The Dragon Weapons Set is a Weapon Set in Fortnite: Save the World that was introduced in Update v2.50.

It contains 3 Melee Weapons and 5 Ranged Weapons.

Obtained

Obtained[]

Certain Schematics from this set can be obtained from Lunar Llamas, where as the rest can be purchased from the Event Store for 1,680 Gold each.

Schematics available from the Lunar Llamas can be researched with Epic and Legendary Flux, and Weapon Designs.

Schematics available from the Event Store can be researched with a Weapon Research Voucher each.
