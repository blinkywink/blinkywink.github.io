The Vindertech Weapon Set is a Weapon Set in Fortnite: Save the World that was introduced in Update v1.9.1.
Schematics from this set can be obtained from Mission Alerts, Llama Pinatas and Quests.

It contains 3 Melee Weapons and 7 Ranged Weapons.

Trivia

Trivia[]

  • All Ranged Weapons in this set except the Vindertech Seeker use Energy Ammo.
  • This is one of the most expensive sets to craft, since weapons in it require Batteries along with the standard Crafting Resources to craft.
