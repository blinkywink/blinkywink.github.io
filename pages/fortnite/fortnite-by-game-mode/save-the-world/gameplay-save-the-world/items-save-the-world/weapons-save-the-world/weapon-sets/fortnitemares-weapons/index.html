Overview


The Fortnitemares Weapons Set is a Weapon Set in Fortnite: Save the World.
Certain Schematics from this set can be obtained from Fortnitemares Llamas, some from Event Quests, and the rest from Event Shop.

It contains 2 Melee Weapons and 8 Ranged Weapons.
