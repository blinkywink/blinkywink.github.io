Overview


The Rat Rod Weapons Set is an Expansion Weapon Set in Fortnite: Save the World that was introduced in Update v6.00.

It contains 3 Melee Weapons and 6 Ranged Weapons.

Obtained

Obtained[]

Schematics from this set may be made available for Gold in the Weekly Shop.

All items from this set can be researched with Flux and Weapon Designs.
