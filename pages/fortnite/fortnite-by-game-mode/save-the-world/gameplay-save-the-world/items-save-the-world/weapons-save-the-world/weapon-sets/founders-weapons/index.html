Overview


The Founder's Weapons Set is a Weapon Set in Fortnite: Save the World.

It contains 5 Melee Weapons and 11 Ranged Weapons.

Overview

Overview[]

Weapons from the Founder's Weapons Set cannot be researched, collected, recycled, or be found in the Collection Book.

Founder's Llamas consisted of reskins of base game weapons and were exclusively available in Epic rarity only, and which could not be increased. These weapons have a core perk (6th perk) available, which was special since mostly only Legendary weapons had core perks, until the introduction of Black Metal Weapon Set in 2019.

Obtained

Obtained[]

Schematics from this set could only be obtained by purchasing different tiers of Founder's Packs.
