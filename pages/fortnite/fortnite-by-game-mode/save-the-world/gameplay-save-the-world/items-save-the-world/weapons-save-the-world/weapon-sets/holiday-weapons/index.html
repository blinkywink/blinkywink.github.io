Overview


The Holiday Weapon Set is a Weapon Set in Fortnite: Save the World.
Certain Schematics from this set can be obtained from the Frostnite Quests, some from the Holiday Llamas, and the rest from the Event Store, during the Frozen Fjords Venture Season.

It contains 4 Melee Weapons and 15 Ranged Weapons.
