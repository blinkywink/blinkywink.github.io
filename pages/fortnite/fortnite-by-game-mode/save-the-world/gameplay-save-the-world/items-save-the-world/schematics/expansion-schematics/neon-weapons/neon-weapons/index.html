Overview


The Neon Weapons Set is a Weapon Set in Fortnite: Save the World that was introduced in Update v3.50.

It contains 3 Melee Weapons and 7 Ranged Weapons.

Obtained

Obtained[]

Schematics from this set can be obtained from Mission Alerts, Llama Pinatas and Quests.

All items from this set can be researched with Epic and Legendary Flux, and Weapon Designs.

Trivia[]
