Overview

This article is about the Save the World version. For the Battle Royale version, see Vending Machine (Battle Royale).

“
Clip’s made some major field repairs and fixed up the Vending Machines!
”
— Homebase Status Report 7.21.2020

Vending Machines are objects in Fortnite: Save the World that are exclusive to Ventures. They were introduced alongside Ventures in Update v13.30.

There can be up to four Vending Machines and two Repair Benches in each mission of Ventures.

Functionality

Functionality[]

To obtain a weapon from Vending Machines, players must spend crafting materials gathered during their missions. Each Vending Machine offers three weapons, which can be cycled by striking the machine with a pickaxe. Only one weapon can be obtained per Vending Machine before it disappears.

The item pool available from Vending Machines varies by season, while items listed under All Seasons remain available at all times.
