Overview

This article is about the Save the World Questline. For the LTM, see Impostors.

“
Search the newly discovered Blasted Badlands for a true Hero! Train Dennis to become a warrior in the new Blasted Badlands Venture Zone to unlock Homebase's own burger boss!
”
— Event Description

Imposter! is an Event Questline in Fortnite: Save the World that was first added in Update v13.30. It is available during the Blasted Badlands Venture Season, and rewards players with 3,250 Road Trip Tickets, a Core RE-PERK!, and a Legendary Dennis.

It has a total of 4 Pages with 15 Quests total.

Lore

Lore[]

The Blasted Badlands are discovered and Dennis confronts his Imposter Syndrome, which starts to haunt him just as his Hero training with Major Oswald begins. To support him, the Crew assigns Dennis easier tasks to boost his confidence, while the Commander scours the zone for CCTV security footage to trace the source of the Imposter. Dennis is then given various assignments, including surveying the area, smashing old cars, rescuing Survivors, and eliminating Husks. As Dennis starts to believe in himself and gain confidence, the Imposter is finally defeated and fades away.

Trivia

Trivia[]

  • This was the first Event Questline with Quests requiring completion in a Venture Zone.
  • This is also the first Event Questline to reward players with Core RE-PERK!.
    • The only other source for them currently is Venture level-up rewards, Venture Quests and the Weekly 160s Quest.
    • The Baddest Lands Quest used to offer a Weapon Supercharger instead of a Core RE-PERK!
