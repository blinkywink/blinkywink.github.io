Overview

“
A blast from the past! Guarantees a weapon schematic and hero introduced during Fortnite events. One Epic or better, and one Legendary!
”
— In-Game Description

Birthday Llamas are a type of Event Llamas in Fortnite: Save the World that can be obtained as part of the Fortnite Birthday Celebration Event, available during the Blasted Badlands Venture Season.

List

List[]

Birthday Llamas contain all Schematics and Heroes from the following:

  • Core Game Weapons Set
  • Event Sets
  • Expansion Sets

They do not include the following:

  • Birthday Heroes
  • Starter Pack Heroes
  • Founder's Pack Heroes and Schematics
  • Xbox Live Heroes
  • PlayStation Plus Heroes
Obtained

Obtained[]

1x Birthday Llama is available to be claimed from the Llama Shop for free until the event ends, with more being obtainable from Fortnite Birthday Celebration Quests.
