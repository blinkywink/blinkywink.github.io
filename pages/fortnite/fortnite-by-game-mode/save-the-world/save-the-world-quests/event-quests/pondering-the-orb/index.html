Mission[]

“
A giant Orb has appeared in the sky above Homebase, and we're getting reports it appears anywhere we operate! Place some scanners to gather intel in any zone!
”
— In-Game Description

Pondering The Orb is an Event Quest in Fortnite: Save the World. It is the 1st Quest, on Page 1 of the A Voyager's Arrival Quests, and unlocks the Droning On Quest.
