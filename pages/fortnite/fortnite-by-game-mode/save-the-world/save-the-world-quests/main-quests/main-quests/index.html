Main Quests

Overview

Main Quests, also known as Campaign Quests, are a set of Quests in Fortnite: Save the World. They require the player to complete various objectives in specific Missions. Main Quests follow along the storyline in a cohesive manner, and feature a range of Characters, almost always including Ray.
