Wargames: Daily Challenges

Wargames: Daily Challenges

Description

Mode

Save the World

Introduced

May 9th 2019

Overview

Wargames: Daily Quests are a set of Quests in Fortnite: Save the World. They reward players with varying rewards depending on the zone, along with a one-time daily grant of Battle Pass XP from the first Quest every day.

Quests[]

Daily Wargames
Eligible quests are located under the Wargames: Daily Challenges category of the Quest Log.
