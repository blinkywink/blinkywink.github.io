Overview

Locations in Fortnite: Save the World include 4 Base Game Locations and 5 Venture Locations, each with its own storyline except for Twine Peaks.

Base Game Locations are unlocked by completing the Launch the Rocket Side Quest in the previous zone, with each new zone increasing in difficulty and requiring a higher Power Level.

Event Locations are accessible to all players who own Save the World and feature missions of all Power Levels. Some of these zones also contain Ventures.
