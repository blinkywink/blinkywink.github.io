Stonewood is the first location that players progress through in Fortnite: Save the World . It has a recommended Power Rating of 1 to 19. Players will need to complete all the Stonewood Main Quests before they're able to unlock Plankerton.

Copper Ore is the primary ore in Stonewood, Players are able to find silver ore in higher-end missions.

Players will learn to build structures, place traps and defend different types of objectives from waves of husks. New enemy types are gradually introduced through Storm Shield Defenses. This is the starting zone for all players, so there are several tutorial quests available to complete.
