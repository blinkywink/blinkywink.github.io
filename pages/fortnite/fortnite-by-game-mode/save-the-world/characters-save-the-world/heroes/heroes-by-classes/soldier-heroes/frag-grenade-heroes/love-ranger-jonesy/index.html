Overview

This article is about the Save the World Hero. For the Battle Royale Outfit, see Love Ranger.

“
Aim for the heart.
”
— In-Game Description

Love Ranger Jonesy is a Legendary Soldier in Fortnite: Save the World that can be purchased from the Event Store for 2,800 Gold during the Flannel Falls Venture Season. He is a part of the Springtime Heroes Set.

Power Impact

Power Impact +

Increases Shockwave Radius by 112.5%.

Replaces the Standard Perk if this Hero is slotted as the Commander.
