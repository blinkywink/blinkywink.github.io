Overview

This article is about the Hero in Save the World. For the Outfit in Battle Royale, see Calamity.

“
Fastest draw in the west.
”
— In-Game Description

Quickdraw Calamity is a Mythic Soldier in Fortnite: Save the World that can be obtained from Wild West Llamas. She is a part of the Wild West Heroes Set.

Perks

Perks[]

Hero Perk[]

STANDARD PERK
Six Shooter

Increases Pistol Damage by 20% and Impact by 25% for the first 6 shots in magazine.

Passive bonus granted if this Hero is slotted in a Hero Loadout.
 
COMMANDER PERK
Six Shooter +

Increases Pistol Damage by 40% and Impact by 50% for the first 6 shots in magazine.

Replaces the Standard Perk if this Hero is slotted as the Commander.

Team Perk[]

Hot Swap
Requires:
One Hero of each Class
Weapon eliminations increase Weapon Damage by 13% for 15 seconds. Each weapon type can only apply one stack of this effect.
