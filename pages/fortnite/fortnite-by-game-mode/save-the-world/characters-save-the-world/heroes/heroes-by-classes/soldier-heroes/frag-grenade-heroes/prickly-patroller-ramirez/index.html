Overview

This article is about the Save the World Hero. For the Battle Royale Outfit of a similar name, see Prickly Patroller.

“
Just a cactus being a cactus.
”
— In-Game Description

Prickly Patroller Ramirez is a Legendary Soldier in Fortnite: Save the World. She can be unlocked by completing the Ricochet Ranger Quest in the Raptor Wrangler Questline, available during the Blasted Badlands Venture Season. She is a part of the Road Trip Heroes Set.

Perk

Perk[]

STANDARD PERK
Prickly

Enemies that land melee attacks take 13 Base Damage.

Passive bonus granted if this Hero is slotted in a Hero Loadout.
 
COMMANDER PERK
Prickly +

Enemies that land melee attacks take 24 Base Damage plus an additional 13 Base Damage per second for 3 Seconds.

Replaces the Standard Perk if this Hero is slotted as the Commander.
