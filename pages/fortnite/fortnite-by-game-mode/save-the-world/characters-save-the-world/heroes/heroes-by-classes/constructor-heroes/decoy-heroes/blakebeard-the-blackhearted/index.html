Overview

This article is about the Save The World Hero. For the Battle Royale Outfit, see Blackheart.

“
The dreaded captain of the stormy seas
”
— In-Game Description

Blakebeard the Blakehearted is a Mythic Constructor Hero in Fortnite: Save the World that can be obtained by completing the SEE-Wolf Quest in the Yarrr! Questline. He is a part of the Pirate Heroes Set, and grants access to the Blakebeard's Stash Team Perk.

Perk

Perk[]

STANDARD PERK
Avast Mateys

Increases Cannonball Damage by 20%.

Passive bonus granted if this Hero is slotted in a Hero Loadout.
 
COMMANDER PERK
Avast Mateys +

Increases Cannonball Damage by 40%. Converts R.O.S.I.E to fire 30 cannonballs.

Replaces the Standard Perk if this Hero is slotted as the Commander.


Cannonballs affected by this Perk are Cannonball Grenade, Bowler, Cannonade and Jabberwocky. It does not affect the Broadside Trap.
