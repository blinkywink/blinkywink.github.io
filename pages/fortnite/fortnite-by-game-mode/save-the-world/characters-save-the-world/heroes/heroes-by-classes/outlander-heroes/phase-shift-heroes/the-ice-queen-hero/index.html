Overview

This article is about the Save the World Hero. For the Battle Royale Outfit of the same name, see The Ice Queen.

“
Long live the queen.
”
— In-Game Description

The Ice Queen is a Legendary Outlander in Fortnite: Save the World. She can be purchased from the Event Store for 2,800 Gold during the Frozen Fjords Venture Season. She is a part of the Holiday Heroes Set.
