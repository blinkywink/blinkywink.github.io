Overview

This article is about the character from Save the World. For the Battle Royale Recruit, see Recruit Jonesy.

“
I don't know what I've been told, something... Something... Whatever, I'm great!
”
— Jonesy

Jonesy is a Character in Fortnite: Save the World, and a Soldier as a Hero in-game.

Description

Description[]

Jonesy is a male with blond hair, and blue eyes.

Lore

Lore[]

Jonesy was revealed to be in a relationship with Sarah during Love Storm in 2019.
