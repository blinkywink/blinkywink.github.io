Spitfire is a Character in Fortnite: Save the World, and a Soldier as a Hero in-game.

Description

Description[]

Spitfire is a male with short black hair, a beard and brown eyes.

Lore

Lore[]

During the Blockbuster questline, Spitfire and Lok investigated the meteor landing by collecting meteorite shards and placing scanners. It is later revealed that Spitfire left Homebase after becoming ill with radiation sickness.

After searching through various Vindertech consoles, Ray discovered plans regarding the failed 'Project Tailor' which sought to create an exosuit that if built would've been able to keep Spitfire alive. After having successfully created and tested the suit, Spitfire returned to Homebase as the superhero Carbide.
