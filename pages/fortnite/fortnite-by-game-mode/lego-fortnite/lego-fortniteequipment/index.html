LEGO Fortnite:Equipment

Overview

Equipment are items that will give players an advantage in LEGO Fortnite. Equipment is craftable throughout the game using the Crafting Bench.

Overview

Overview[]

Equipment is a strictly defined group of items that can be used to make certain tasks easier throughout the game, travel faster, modify items, decrease damage taken, and other certain effects.

An item is classified as Equipment if they are in that tab in the Sandbox version of LEGO Fortnite.
