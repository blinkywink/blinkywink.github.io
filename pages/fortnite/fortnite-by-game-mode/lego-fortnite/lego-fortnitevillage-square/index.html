Village Square

Overview


A Village Square is a structure in LEGO Fortnite that attracts NPC Villagers to help you throughout the game.

Types

Types[]

There are multiple types of Village Squares, being different in each biome. The visuals and models stay the same throughout each upgrade but the benefits have slight differences. See below.
