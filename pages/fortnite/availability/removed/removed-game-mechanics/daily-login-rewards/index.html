Daily Login Rewards

Overview

This page is about an item or mechanic that has been removed from the game.
This page is under construction.
Please refrain from editing the page if another user is actively doing so as to avoid edit conflicts.
“
With the release of Fortnite’s v25.10 update we’ll be making a significant change to the reward systems in Save the World. We’ll be removing the Daily Login Rewards, and shuffling those rewards to other sources — most notably Daily Quests and Mission Alerts.
”
— Update v25.10

Daily Login Rewards were an exclusive system in Fortnite: Save the World that rewarded players daily for logging in. The cycle of rewards looped upon reaching 336 logins. The system encouraged total login times and not consecutive login times, so a player would not revert to Day 1 if they missed a day. They were removed in Update v25.10, but other sources for rewards were improved to compensate for for this change. Below is the list of changes made to make-up for the lack of Daily Login Rewards:

Changes to Rewards
General Changes
  • +30 V-Bucks and X-Ray Tickets received from Daily Quests.
  • +10 Gold received from Daily Quests.
  • Increased amount of Evolution Materials sold in the Weekly Store.
  • Endurance Daily Quests now grant Evolution Materials, Perk Materials, and Gold upon completion.
  • Added Rare, Epic, and Legendary items to Campaign Quests in Stonewood and Plankerton to aid out newer players.
All Mission Alert Changes
  • Increased average Rarity of reward items.
  • Increased quantities of reward items.
  • Increased quantity of Evolution Materials earned when you complete a Mission Alert.
  • Increased chance for mission alerts to contain a second reward item of same type and rarity.
  • Mini Reward Llamas now have a chance to be replaced by Upgrade Llama Tokens instead.
  • Added a chance to grant items and Llamas, instead of always rewarding Evolution and Perk Materials.
