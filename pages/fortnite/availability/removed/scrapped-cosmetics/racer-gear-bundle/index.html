The Racer Gear Bundle was a Rare Item Shop Bundle in Fortnite. The Racer Gear Bundle contained the male Outfits from the F1 Racers Set.

Trivia

Trivia[]

  • Racer Gear Bundle was removed from the files in Update v28.01.
  • The F1 Racers Set was added in Update v27.11 in an encrypted pak.[1] Dataminers gained access to the pak at the start of Chapter 5: Season 1 by using an old version of Fortnite on a private server to gain access to the removed files after the AES key was accidentally published by Epic Games.[2]
    • The set was most likely added to the files too early but there’s always the possibility that the entire set was unfinished or scrapped.
