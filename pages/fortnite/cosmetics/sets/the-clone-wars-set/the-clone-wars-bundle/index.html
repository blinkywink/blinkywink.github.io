The Clone Wars Bundle is a Star Wars Series Item Shop Bundle in Fortnite that can be purchased in the Item Shop for 2,700 V-Bucks. The Clone Wars Bundle was first released in Chapter 7: Season 2 and contains cosmetics from The Clone Wars Set.
