Overview

The Order Bundle is a Rare Item Shop Bundle in Fortnite, that can be purchased in the Item Shop for 1,500 V-Bucks. The Order Bundle was first released in Chapter 3: Season 2 and contains The Order's cosmetics from the The Seven Set.
