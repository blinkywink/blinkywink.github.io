Overview

This article is about the original Chapter 2: Season 5 Set. For the released Chapter 5: Season 1 Set, see Family Guy Set.

FrenchFry was a Set in Fortnite related to a planned collaboration with the animated sitcom Family Guy. FrenchFry was intended to release in Chapter 2: Season 5, but was scrapped for unknown reasons.

Trivia

Trivia[]

  • The Disney Megaleak in 2024 confirmed that the FrenchFry Set was originally intended to release in Chapter 2: Season 5[1].
  • FrenchFry was added in Update v15.40.
    • Super Cab-Masher's files contained several frames from episodes of the show, and that were related to the FrenchFry Set; suggesting that it would have had a Back Bling reactive the same way as the former.
    • The default strike effects used in cel-shaded Pickaxes were added through files relating to one made for the FrenchFry Set.
    • IDs for two Pickaxes were found, Scratch and Curly. The former was likely The Scratcher and the latter the cello from The Giant Chicken's concept art (with "curly" possibly referring to the curly maple used in instruments), which was mentioned in the pitch for what would become the Drumsticks.
    • In the State of Unreal 2022 livestream, a file named "FamilyGuy" could be seen alongside "C3S2_ArmTease", "C3S3_JONES", "C3S3_VADER" and "Doom".
  • In Chapter 5: Season 1, The Peter Griffin outfit, alongside the Family Guy Set would be released, with the Back Bling Clip Show Carry-All from the set using a modified version of the texture found prior to the season's debut.
