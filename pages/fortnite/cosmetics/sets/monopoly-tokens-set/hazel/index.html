Hazel is a Rare Back Bling in Fortnite that can be purchased in the Item Shop with the Monopoly Tokens Bundle for 800 V-Bucks. Hazel was first released in Chapter 2: Season 8 and is part of the Monopoly Tokens Set.

Hazel can also be obtained by redeeming a code found in a copy of the Monopoly: Fortnite Collector's Edition board game.
