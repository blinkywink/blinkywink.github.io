Overview

The media represented on this page has been created by a second/third party.
This content is not originally part of the game but has been introduced through a special collaboration.

The Shaq Bundle is an Icon Series Item Shop Bundle, that can be purchased in the Item Shop for 3,400 V-Bucks. The Shaq Bundle was first released in Chapter 5: Season 4 and contains cosmetics from the Shaquille O'Neal Set.
