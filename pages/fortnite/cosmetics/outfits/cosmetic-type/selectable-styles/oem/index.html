Overview

Obtaining this cosmetic also makes it available in Rocket League.

“
Nothing wrong with a reliable set of wheels
”
— In-Game Description

OEM is a Common Wheel in Fortnite, that comes free with every account. OEM was first released in Chapter 5: Season 1.

How To Unlock?[]

  • Black - Achieve Diamond Rank 3 (Season 0 Ranked Quests)
  • Burnt Sienna - Achieve Diamond Rank 2 (Season 0 Ranked Quests)
  • Forest Green - Achieve Platinum Rank 3 (Season 0 Ranked Quests)
  • Lime - Achieve Silver Rank 3 (Season 0 Ranked Quests)
  • Pink - Achieve Diamond Rank 1 (Season 0 Ranked Quests)
  • Saffron - Achieve Gold Rank 3 (Season 0 Ranked Quests)
  • Gray - Achieve Gold Rank 2 (Season 0 Ranked Quests)
  • Orange - Achieve Bronze Rank 2 (Season 0 Ranked Quests)
  • Cobalt - Achieve Silver Rank 1 (Season 0 Ranked Quests)
  • Sky Blue - Achieve Platinum Rank 1 (Season 0 Ranked Quests)
  • Titanium White - Achieve Elite Rank (Season 0 Ranked Quests)
  • Crimson - Achieve Champion Rank (Season 0 Ranked Quests)
  • Purple - Achieve Unreal Rank (Season 0 Ranked Quests)

or Earn any color from Drops or by purchasing them in the Store in Rocket League
