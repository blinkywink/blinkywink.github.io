Outfit[]

“
Real queens take what they want.
”
— In-Game Description

Dylan is an Epic Outfit in Fortnite that can be obtained by purchasing all cosmetics on Page 1 of Dylan's Set in the Chapter 7: Season 3 Battle Pass, which can only be unlocked after reaching Level 60. Dylan is part of the Mercenary Heartbreaker Set.
