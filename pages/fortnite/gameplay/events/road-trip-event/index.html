Overview

This article is about the Event in Save the World. For the other uses of "Road Trip", see Road Trip (disambiguation). For the Main Quest in Save the World, see Road Trip (Quest).
“
Tell me, partner...do ya feel lucky? Test that luck with the new Flintlock weapon set while you Challenge the Horde in the Road Trip event, now available in Save the World.
”
— v5.00 Patch Notes

Road Trip was an Event in Fortnite: Save the World that was introduced in Update v5.00. It included a Questline, the introduction of Wild West Heroes and the Flintlock Weapons Set, as well as the reintroduction of the Challenge the Horde mode.
